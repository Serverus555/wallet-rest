## config.env
* ### Прописаны дефолтные настройки
* ### CACHE_TYPE - кэш для запросов на получение баланса
   * noop - выкл, memory - вкл
   * CACHE_CAPACITY и CACHE_TTL необязательны при noop
   * Не гарантирует валидность данных в конкретный момент времени. Используется только для представления
   * Используется при частых последовательных read. При read/write 1 к 1 по конкретному wallet не даёт прироста
## Конкурентность
* Все запросы к базе выполняются без блокировок и транзакций.
* Потокобезопасность достигается за счёт атомарности INSERT/UPDATE.
